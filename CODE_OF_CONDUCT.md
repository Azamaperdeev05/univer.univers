# Code of Conduct

Univer Platonus жобасының мақсаты - студенттерге пайдалы, қауіпсіз және түсінікті құрал жасау. Жобаға қатысатын әр адамға құрметпен қарау міндетті.

## Біздің міндетіміз

Біз ашық, сыпайы және кәсіби орта құруға тырысамыз. Қатысушылар тәжірибесіне, тіліне, ұлтына, жасына, жынысына, дініне, көзқарасына немесе техникалық деңгейіне қарамастан құрмет көруі керек.

## Қабылданатын мінез-құлық

- Сыпайы және нақты сөйлеу.
- Сын айтқанда шешім ұсыну.
- Жаңадан келген қатысушыларға көмектесу.
- Басқа адамның уақытын құрметтеу.
- Қате тапқанда адамды емес, мәселені талқылау.
- Құпия деректерді жарияламау.

## Қабылданбайтын мінез-құлық

- Қорлау, кемсіту, жеке шабуыл.
- Қудалау немесе қысым көрсету.
- Рұқсатсыз жеке ақпарат жариялау.
- Құпия сөз, token, cookie немесе student data жариялау.
- Әдейі жалған ақпарат тарату.
- Жобаға зиян келтіретін әрекеттер.

## Maintainer жауапкершілігі

Maintainer-лер:

- Ережені түсіндіреді.
- Қақтығыстарды әділ қарайды.
- Қажет болса comment, issue немесе PR-ды жабады.
- Қайталанған бұзушылық болса қатысуды шектейді.
- Қауіпсіздікке қатысты деректерді public талқылаудан алып тастайды.

## Қолдану аясы

Бұл ереже:

- GitHub Issues.
- Pull Requests.
- Discussions.
- Commit messages.
- Telegram/community арналар.
- Жобаны таныстыру немесе қолдау кезіндегі public кеңістік.

## Хабарлау

Ереже бұзылғанын байқасаңыз, maintainer-ге жеке хабарласыңыз немесе Issue ашыңыз. Қауіпсіздік немесе құпия дерекке қатысты мәселе болса, public Issue ашпай, [SECURITY.md](SECURITY.md) нұсқаулығын қолданыңыз.

Хабарламада:

- Не болды.
- Қай жерде болды.
- Қатысты сілтеме немесе скриншот.
- Қандай әрекет күтіледі.

## Шешім қабылдау

Әр жағдай контекст бойынша қаралады. Мақсат - жазалау емес, жобаны және қатысушыларды қорғау.

Мүмкін шаралар:

- Ескерту.
- Comment/PR/Issue өңдеу немесе жабу.
- Уақытша шектеу.
- Тұрақты шектеу.

## Дереккөз

Бұл құжат Contributor Covenant принциптеріне сүйенеді, бірақ жоба контекстіне бейімделген.
